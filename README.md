# 🐍 Flutter Snake Game

A simple Snake Game built with Flutter and a custom joystick.  
Move the snake, eat the food, and score as much as you can! 🎮

---

## 🚀 Features

- 🎛️ Custom joystick control
- 🐍 Snake movement
- 🍎 Food spawn and eating
- 📊 Score counting
- 💀 Game over dialog with restart option
